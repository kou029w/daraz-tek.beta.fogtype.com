# daraz-tek.beta.fogtype.com

だらずてっくの Slack のアーカイブ

Slack のログの移行方法: <https://docs.mattermost.com/onboard/migrate-from-slack.html>

## 構成

- Ubuntu 22.04 LTS Server

## 使用方法

### サーバーへのインストール

```
make install
```

### サーバーにデプロイ

```
make deploy
```

### 環境変数ファイルの生成

```
make .env
```

## ライセンス

WTFPL
